What I learn?

JavaScript

There are different opinions about JavaScript. For me, this language is very intuitive. Even if I have a problem, I can usually solve it very quickly with the help of a friendly browser console.

React

Initially, the React library seemed a bit complicated to me, but I changed my mind very quickly. What surprised me the most was the almost limitless possibilities of creating components and the fully automated applications rendering.

JQuery

I can use the most important functions of the JQuery library. To build more extensive applications, I think the React library is more valuable to learn, but JQuery is still great to create simpler applications or interactive websites - especially when there is a need for frequent communication with the DOM.

HTML

At the moment I know the HTML syntax at a good level. I try to write website structures in accordance with HTML5 standards. I use semantic tags to build layouts. Currently, I learn how to add classes according to the BEM method.

CSS

CSS is a technology that will probably always surprise me with new solutions. Recently, I have learned how to use animations, how to build page elements using flexboxes and the basics of CSS grid. Creating this website was a very good practice for me of these functions.

SASS

I have learned the SASS preprocessor some time ago, but in the SCSS syntax, because indentation syntax had always been for me less transparent than the brace syntax. However, I recently learned the syntax of "pure" SASS, which I also practiced writing this website.

Bootstrap

Currently, more than half of visits to websites are visits from mobile devices. That is why I recently took a lot of time to learn issues related to RWD. Bootstrap is a great help in creating responsive websites and that is why I currently learn using this framework, but I have not used it yet in projects.

PHP

At the moment, I only know the basics of PHP, because I am at the beginning of my adventure with programming, and at the moment I focused primarily on frontend technologies.

My projects

Weather Forecast App

JS (ES6) React JSX API CSS3

A fully responsive application that displays information about the current and forecast weather for many cities in Poland, based on data from openweathermap.org. Application developed in the "create-react-app" environment, styled according to the "mobile first" rule.

Currency Calculator App

JS (ES6) React JSX API CSS3

The application displays the current exchange rates of the most popular currencies in Poland and it also allows you to convert values between selected currencies, based on data from the NBP. Application developed in the "create-react-app" environment, using React Router components, in the form of Single Page Application.

Business card website

HTML5 CSS3 SASS JS (ES6) JQuery

This is my latest business card in the form of a website. By the way, you are just on this website. After learning animation in CSS, flexboxes and JQuery, I decided to practice my newly acquired skills. I styled the application in the syntax of "pure" SASS, according to the "mobile first" rule.

Check Number App

JS (ES6) React JSX CSS3

The application is used to check the correctness of selected numbers in accordance with the standards adopted in Poland. With this application you can quickly verify the correctness of the following numbers: pesel, NIP, REGON, ID-card number and bank account number. Application developed in the "create-react-app" environment, using React Router components, in the form of Single Page Application.

Amount in Words App

JS (ES6) React JSX CSS3

The application is used to convert the entered amount to a word form, which is very often used in invoice and bill templates. The application was developed in the "create-react-app" environment and the data was prepared by me in a separate file in JSON format.
