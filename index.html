Czego się uczę?

JavaScript

Na temat języka JavaScript są różne opinie. Dla mnie język ten jest bardzo intuicyjny. Nawet jeśli napotkam jakiś problem to zazwyczaj bardzo szybko jestem w stanie go rozwiązać z pomocą równie przyjaznej konsoli w przeglądarce.

React

Początkowo biblioteka React wydawała mi się trochę skomplikowana, jednak bardzo szybko zmieniłem co do niej zdanie. Najbardziej zaskoczyły mnie niemal nieograniczone możliwości tworzenia komponentów i w pełni zautomatyzowane renderowanie aplikacji.

JQuery

Potrafię posługiwać się najważniejszymi funkcjami biblioteki JQuery. Budując rozbudowane aplikację uważam, że bardziej wartościowa do nauki jest biblioteka React, jednak JQuery nadal świetnie się sprawdza przy budownie prostszych aplikacji, czy interaktywnych stron - zwłaszcza takich, w których występuje konieczność częstej komunikacji z DOM.

HTML

Składnię języka HTML na tą chwilę umiem na dobrym poziomie. Staram się pisać strukturę stron, zgodnie ze standardami HTML5. Do budowy layoutów używam znaczników semantycznych. Aktualnie uczę się nadawania klas, zgodnie z metodą BEM.

CSS

CSS jest technologią, która prawdopodobnie zawsze będzie mnie zaskakiwała nowymi rozwiązaniami. W ostatnim czasie nauczyłem się korzystania z animacji, budowy elementów strony z wykorzystaniem flexboxów oraz podstaw CSS grid. Tworzenie tej strony było dla mnie bardzo dobrym ćwiczeniem tych funkcji.

SASS

Preprocesora SASS uczyłem się już kiedyś, ale w składni SCSS, ponieważ składnia w formie wcięć zawsze przegrywała u mnie ze składnią klamrową. Niedawno jednak przekonałem się do składni "czystego" SASSa, co również przećwiczyłem pisząc tą stronę.

Bootstrap

Obecnie więcej niż połowa wejść na strony internetowe to wejścia z urządzeń mobilnych. Dlatego w ostatnim czasie dużo czasu poświęciłem na naukę zagadnień, związanych z RWD. Bootstrap stanowi duże ułatwienie w tworzeniu responsywnych stron, dlatego aktualnie uczę się korzystać z tego frameworka, jednak jeszcze nie korzystałem z niego przy tworzeniu projektów.

PHP

Na tą chwilę znam jedynie podstawy języka PHP, ponieważ jestem dopiero na początku swojej przygody z programowaniem i dlatego w tej chwili skupiłem się przede wszystkim na technologiach frontendowych.

Moje projekty

Aplikacja Prognoza Pogody

JS (ES6) React JSX API CSS3

W pełni responsywna aplikacja, wyświetlająca informacje o aktualnej i prognozowanej pogodzie dla wielu miejscowości w Polsce, na podstawie danych ze strony openweathermap.org. Aplikacja opracowana w środowisku "create-react-app", stylowana zgodnie z zasadą "mobile first".

Aplikacja Kalkulator Walut

JS (ES6) React JSX API CSS3

Aplikacja wyświetlająca dane o aktualnych kursach najpopularniejszych w Polsce walut wraz z możliwością przeliczenia wartości pomiędzy wybranymi walutami, na podstawie danych ze strony NBP. Aplikacja opracowana w środowisku "create-react-app", z wykorzystaniem komponentów React Router, w formie Single Page Application.

Strona internetowa - wizytówka

HTML5 CSS3 SASS JS (ES6) JQuery

Jest to moja najnowsza wizytówka w formie strony internetowej. Swoją drogą, właśnie znajdują się Państwo na tej stronie. Po nauce animacji w CSS, flexboxów oraz JQuery, postanowiłem przećwiczyć nowo nabyte umiejętności. Stylowanie aplikacji wykonałem w składni "czystego" SASSa, zgodnie z regułą "mobile first".

Poprzednia wersja strony

HTML5 CSS3 JS (ES6)

Poprzednia wersja mojej strony internetowej, która stanowi wierne odwzorowanie mojego CV. Po nauce nowych funkcjonalności CSS postanowiłem napisać nową stronę, na której aktualnie się Państwo znajdujecie.

Strona internetowa reklamująca napoje smoothie

HTML5 CSS3 SCSS JS (ES6)

Strona internetowa, reklamująca linię napojów smoothie dla fikcyjnej firmy. Większa część tekstu została zastąpiona fragmentami "Lorem ipsum", ponieważ podstawowym celem tego projektu było przedstawienie aspektów wizualnych.

Co jeszcze chcę poznać?

Redux

Biblioteka React zaciekawiła mnie tak bardzo, że w tej chwili Redux jest numerem 1 na mojej liście technologii, których chcę się w niedalekiej przyszłości nauczyć.

Node JS

W tej chwili skupiłem się przede wszystkich na nauce technologii frontendowych, jednak w przyszłości chciałbym umieć pracować również przy backendzie aplikacji. Z racji mojej sympatii do JS i React, chciałbym się nauczyć również pracy z NodeJS.

Angular

Angulara w tej chwili nie znam, jednak od wielu osób słyszałem, że jest to bardzo dobry framework do pracy przy większych projektach, jednak jest również trochę trudniejszy niż React. Dlatego też naukę rozpocząłem od Reacta, ale Angulara również chciałbym się w przyszłości nauczyć.
